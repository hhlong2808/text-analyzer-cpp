# Text Analyzer (C++)

A simple C++ project that analyzes text files and provides:

- Word count
- Word frequency
- Top 10 most frequent words
- Longest word
- Longest substring without repeating characters (Sliding Window)

## Technologies Used
- C++
- STL (unordered_map, vector, algorithm)
- File I/O

## How to Run

Compile:

g++ main.cpp -o analyzer

Run:

./analyzer

Make sure input.txt exists in the same folder.
